================================================
    HOW TO USE THE DOC NUMBER FILTER  💙
================================================

STEP 1:  Put your Word documents (.docx) into the "input" folder.

STEP 2:  Double-click the file called:
              ▶  Run Filter.command

         (If Mac asks for permission the first time, 
          right-click it → Open → Open)

STEP 3:  Find your cleaned documents in the "output" folder!


================================================
    WHAT THE TOOL DOES
================================================

  ✓  Removes line numbers   (e.g., "1.", "2)", "[3]")
  ✓  Removes timestamps     (e.g., "00:12", "1:23:45")
  ✓  Joins a line to the previous paragraph
     if it starts with a small (lowercase) letter.
  ✓  Starts a new paragraph
     if it starts with a Capital (uppercase) letter.


================================================
    EXAMPLE
================================================

  BEFORE:
    1. Hello my name is John.
    00:12 i am a student
    2. She is very kind.
    3. and helpful too.

  AFTER:
    Hello my name is John. i am a student
    She is very kind. and helpful too.

================================================
